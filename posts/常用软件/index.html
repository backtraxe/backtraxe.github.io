跨平台

  1. Visual Studio Code：代码编辑器。开源 免费
  2. JetBrains Toolbox App：JetBrains IDE 管理器。 学生免费
  3. PDF Reader Pro：PDF 编辑器。 付费

macOS

  1. Homebrew：软件包管理器。 开源 免费
  2. Android File Transfer：和 Android 之间传输文件。 免费
    • brew install android-file-transfer
  3. IINA：视频播放器。 开源 免费
    • brew install iina
  4. OpenInTerminal-Lite：访达插件，在终端/编辑器中打开当前目录。 开源 免费
    • brew install openinterminal-lite
    • brew install openineditor-lite
  5. shottr：截图、滚动截图、抠图、测距、OCR、取色。 免费
  6. maccy：剪贴板历史记录。 免费 开源
    • brew install maccy
  7. MonitorControl：外接显示器亮度控制。 免费 开源
