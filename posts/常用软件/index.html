跨平台

  1. Mark Text：简洁美观的 Markdown 编辑器。开源 免费
  2. Visual Studio Code：代码编辑器。开源 免费
  3. JetBrains Toolbox App：JetBrains IDE 管理器。 学生免费
  4. PDF Reader Pro：阅读、编辑、注释、转换和签署 PDF 的最佳 PDF 软件。 付费

macOS

  1. Homebrew：适用于 macOS（或 Linux）缺失的软件包的管理器。 开源 免费
    • /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Hex Fiend：适用于 macOS 的快速且智能的开源十六进制编辑器。 开源 免费
    • brew install --cask hex-fiend
  3. Mounty for NTFS：在 macOS 下以读写模式重新挂载写保护的 NTFS 卷的小工具。 免费
    • brew install --cask mounty
  4. Android File Transfer：在 Mac 计算机和 Android 设备之间浏览和传输文件。 免费
    • brew install --cask android-file-transfer
  5. IINA：适用于 macOS 的现代媒体播放器。 开源 免费
    • brew install --cask iina
  6. OpenInTerminal-Lite：适用于 macOS 的 Finder 工具栏应用程序，用于在终端、iTerm、Hyper 或 Alacritty 中打开当前目录。 开源 免费
    • brew install --cask openinterminal-lite
    • brew install --cask openineditor-lite
  7. MacZip：专为 macOS 而设计的压缩软件. 免费
    • brew install --cask maczip
  8. CheatSheet：适用于 macOS 的列出当前应用程序所有快捷键的工具。 免费
    • brew install --cask cheatsheet
  9. 腾讯柠檬清理：适用于 macOS 的磁盘清理工具。 免费
  10. QSpace：一款简洁高效的多视图文件管理器。 付费
  11. Alfred：通过自定义操作控制你的 Mac 电脑，提高工作效率。 付费
  12. Downie：适用于 macOS 的视频下载工具。 付费
  13. Dash：API 文档浏览器和代码片段管理器。 付费
  14. shottr：截图、滚动截图、抠图、测距、OCR、取色。 免费
