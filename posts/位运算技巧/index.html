基本位运算

  • &：与。同1则1，其余为0。
  • |：或。同0则0，其余为1。
  • ~：非。01互换。
  • ^：异或。同0异1。
  • <<：左移。右补0。
  • >>：带符号右移。正数左补0，负数左补1。
  • >>>：无符号右移。左补0。

//    -n == (~n) + 1
// -(~n) == (~(~n)) + 1
// -(~n) == n + 1
n = -(~n);

1
2
3
4
5

//    -n == (~n) + 1
// -(-n) == (~(-n)) + 1
//     n == (~(-n)) + 1
// ~(-n) == n - 1
n = ~(-n);

将一个数二进制表示中最低位 1 置为 0

1
2
3
4

//           n == 0b...10...0
//       n - 1 == 0b...01...1
// n & (n - 1) == 0b...000000
n & (n - 1);

只保留一个数二进制表示中最低位 1

1
2
3
4

//                n == 0b.....10...0
//            n - 1 == 0b.....01...1
//      n & (n - 1) == 0b.....000000
// n -= n & (n - 1) == 0b0...010...0
n -= n & (n - 1);
